Skills

技能矩阵与实现状态

当前对外主叙事聚焦 `quant-data`、`quant-backtest`、`quant-report` 三件套；`quant-alpha` 和 `quant-risk` 作为进阶研究增强模块保留，不抢第一入口。

Core

当前主入口

已实现 MVP

quant-data

在线主入口为 `AkShare` 与 `yfinance`，并保留 `demo / csv / local / duckdb` 作为外部兼容输入；支持指标补充、metadata、manifest 与数据修复。

  • 指标：MA、MACD、RSI、Boll、ATR、KDJ、Ret、Momentum、Volatility、ZScore、OBV
  • 输出：CSV / JSON / sidecar meta / manifest / metadata
  • 定位：策略研究前的数据准备与在线取数编排

已实现 MVP

quant-backtest

支持 `sma_cross`、`macd`、`boll_breakout` 三种基础策略，已包含 benchmark、成本模型、trade log、compare / sweep 与 Plotly HTML 报告。

  • 指标：收益、年化、波动率、夏普、最大回撤、胜率
  • 输入：quant-data 结果或在线拉取数据
  • 定位：策略逻辑快速验证

已实现 MVP

quant-report

读取回测 JSON，输出 Markdown / HTML / PDF 报告，统一摘要、风险与改进建议结构，并支持水印控制。

  • 定位：研究结果交付与展示物生成
  • 输出：Markdown / HTML / PDF
  • 适合作为官网和 GitHub 样品主入口

Workflow

典型输入输出

输入

  • 标的与时间区间：如 `000001.SZ`、`2025-01-01 ~ 2025-03-31`
  • 数据源：`akshare / yfinance / demo / csv / local / duckdb`
  • 策略参数：如 `fast=3,slow=8`
  • 报告类型：`markdown / html / pdf`

输出

  • 数据集：CSV / JSON / metadata / manifest
  • 回测：JSON / HTML，含 benchmark 与 trade log
  • 研究报告：Markdown / HTML / PDF
  • Alpha / Risk：JSON / HTML

FAQ

常见问题

是否支持真实 A 股 5 分钟数据？

支持外部兼容输入。当前官网主路径强调在线研究，不承诺项目内保存历史行情；如果你已有 5m 文件或自有 DuckDB，可按兼容输入接入研究链路。

是否直接售卖原始行情数据？

不售卖原始数据。付费价值聚焦于工具能力、研究工作流、报告增强与可复用模板。

是否能做官网展示样品？

可以。仓库已补 `showcase` 样品整理脚本，并继续补统一生成脚本，用于产出回测 HTML、PDF 和 Alpha / Risk 页面样品。

是否提供实盘和投资建议？

不提供。Agent4Quant 的定位是量化研究工具与教程，不涉及荐股、实盘执行或证券咨询。
