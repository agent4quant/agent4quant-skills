案例与展示物

当前案例页优先围绕 `quant-data`、`quant-backtest`、`quant-report` 三件套展开，目标是把“本地研究闭环”讲清楚，而不是把所有模块平均铺开。

Core Skill

quant-data

先解决研究前的数据准备问题：在线取数、兼容输入、指标补充和基础校验。

Core Skill

quant-backtest

把研究数据转成可验证的策略结论，输出 benchmark、成本、trade log 和 Plotly 结果。

Core Skill

quant-report

把结果变成可交付研究报告，输出 Markdown / HTML / PDF 样品，直接进入展示和沟通环节。

Flow

核心三件套闭环

第一步：数据准备

  • `AkShare` 作为 A 股主入口
  • `yfinance` 作为港美股主入口
  • 兼容 CSV / local / DuckDB 自有输入

第二步：策略验证

  • 单策略回测与参数扫描
  • 多策略对比、benchmark、成本模型
  • Plotly HTML 结果和 JSON 摘要

第三步：研究交付

  • 输出 Markdown / HTML / PDF 报告
  • 保留免责声明和水印控制
  • 作为官网、GitHub、咨询沟通样品

案例 A：A 股在线研究准备

使用 `quant-data` 拉取 A 股日线、补技术指标、输出 JSON，适合新用户第一次上手。

案例 B：策略验证与对比

案例 C：研究报告交付

使用 `quant-report` 输出 PDF / HTML 报告，把研究结果变成可发给他人的正式交付物。
